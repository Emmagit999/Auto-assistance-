    Help
    
          
    Getting a key, and linking WhatsApp two ways.
    
        
    
      
    
      
    

        
    
          Setup
          
    Getting an OpenRouter key
    
          
      
            
    1. Open openrouter.ai/keys and sign up — it's free, no card required for free models
      2. 
            
    2. Click Create Key, name it anything (e.g. "androg")
      3. 
            
    3. Copy the key shown — starts with sk-or-v1-, and OpenRouter only shows it once
      4. 
            
    4. Paste it into Androg's setup screen (or the Chat page will prompt you if it's missing)
      5. 
          
    
          
    Free models are rate-limited (20 req/min, 50/day under $10 balance) — Androg automatically rotates across whichever free models are currently available, so a single model being busy won't stop you.
    
        
    

        
    
          WhatsApp
          
    Pairing by QR code
    
          
      
            
    1. Open the WhatsApp tab — it starts trying to connect automatically and shows a QR code
      2. 
            
    2. On your phone: WhatsApp → Settings → Linked Devices → Link a device
      3. 
            
    3. Scan the code on screen with your phone's camera inside that flow
      4. 
          
    
          
    QR codes refresh themselves every so often if you don't scan in time — just wait for the next one, no need to click anything.
    
        
    

        
    
          WhatsApp
          
    Pairing by phone number
    
          
      
            
    1. On the WhatsApp tab, switch to Phone number
      2. 
            
    2. Type your number as country code + number, digits only — e.g. 15551234567 for +1 (555) 123-4567, no +/spaces/dashes
      3. 
            
    3. Tap Get code — an 8-character code appears almost immediately
      4. 
            
    4. On your phone: WhatsApp → Settings → Linked Devices → Link with phone number instead, then type the code in right away — it expires in under a minute
      5. 
            
    5. If it expires before you finish, just tap Get code again for a fresh one
      6. 
          
    
        
    

        
    
          Known issue
          
    "Check the code / number" even though it's right
    
          
    
            If your phone mentions "passkey" anywhere (e.g. "you may need to scan
            another QR code to use your passkey"), this isn't an Androg bug — WhatsApp rolled out a
            new verification step after linking that requires a real biometric/hardware passkey
            ceremony, which no unofficial client (Baileys, whatsapp-web.js, etc.) can complete. It's
            an open, actively-discussed issue upstream, not something fixable from here. Options:
            keep retrying occasionally (it may not affect every account, and a fix may land
            upstream), try a different WhatsApp number, or use WhatsApp's official Cloud API instead
            of this unofficial connection (a bigger setup, but immune to this).
          
    
        
    

        
    
          Good to know
          
    Self vs. dedicated mode
    
          
    
            Self mode (default) links your own everyday WhatsApp number, but the bot only ever
            replies inside your own "Message Yourself" chat — it never jumps into conversations with your real
            contacts. Dedicated mode is for a number that exists purely to be the bot, replying
            to anyone who messages it. Groups are always off until you explicitly add one's JID in the WhatsApp
            tab. Switch modes any time from there.
